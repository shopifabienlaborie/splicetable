Segmented Control

Toggle control for mutually exclusive options, supporting 2-4 segments. Uses the same 3-layer design as Button: Root (gradient frame) → Item outer (elevated surface) → Item inner (content). Position-dependent border radii, hover lift via padding, press flatten with 1px inner offset for tactile feedback. Extracted directly from Figma with 1:1 fidelity.

Prop Type Default Description
items Array<{label, value, iconBefore?, iconAfter?}> [] Array of segment items (2-4 items)
value string undefined Currently selected value
onChange function(value) undefined Called when selection changes
disabled boolean false Whether the control is disabled
fullWidth boolean false Whether control takes full width

React JSX

<SegmentedControl 
  items={[
    { label: 'Option 1', value: 'opt1' },
    { label: 'Option 2', value: 'opt2' }

  onChange={(value) => setSelectedValue(value)}
/>

Usage Guidelines

Button Usage

Use action buttons for primary and secondary actions. Include icons for visual reinforcement of the action. Full-width buttons work best in narrow containers like sidebars.

Segmented Control Usage

Use segmented controls for mutually exclusive options with 2-4 choices. For binary choices (like Fill/Fit), use 2 segments. The active state provides clear visual feedback.

Slider Usage

Sliders are ideal for numeric inputs with a defined range. Always display the current value next to the slider. For percentage values, append the % symbol.

Color Modes

All components automatically adapt to Light and Dark mode based on the user's system preference. Test components in both modes to ensure proper contrast and readability.
